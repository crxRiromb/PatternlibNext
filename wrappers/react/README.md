# React Wrapper for Liebherr2 Patternlib Next

## Create projekt

- Projekt generiert mit: npm create vite@latest wrappers/react
- mit Option: 'TypeScript + SWC'

## Single Build

```bash
yalc link @liebherr2/plnext
npm install
npm run build
```

## Build + Publish via Yalc

```bash
yalc link @liebherr2/plnext
npm install
npm yalc:publish
```
