This Angular app serves as an example application to demonstrate the integration and usage of the Patternlib in an Angular Standalone App. The goal is to provide developers with a quick and easy introduction to using the Patternlib by showcasing practical use cases.

  • Standalone Angular Architecture: The app is based on Angular's new Standalone API and does not require traditional NgModules.
  • Integration of the Patternlib: Demonstrates how Patternlib components can be integrated and used in an Angular application.
  • CSS-Toolkit: The app doesn't use any external CSS toolkit. Instead, it relies on pure CSS and SCSS.
  • Signals: The Signal pattern is used for communication between components, allowing for loose coupling.
    On the Footer page, the footer mode (fixed or scrollable) can be toggled.
    The value is transmitted via a signal to the Shell component, which manages the footer.
  • Translation Library: The app uses the Transloco library for text internationalization.
    Transloco is particularly useful for multilingual applications in the context of Angular Standalone Apps.
    Transloco is integrated into the app and ready for use.
    Note: For efficiency reasons, not all parts of the app are localizedâ€”only the menu.
  • Main Menu: The app includes a main menu that enables navigation between the different pages of the app.
  • Code as a Learning Resource: Developers are provided with an easy-to-understand codebase to help them get started quickly.

This application is ideal for developers who want to familiarize themselves with the Patternlib and efficiently use it in their Angular Standalone Apps.
